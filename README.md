## 📖 项目简介

这是一个专门为儿童设计的贪吃蛇游戏，采用20年儿童游戏开发经验的最佳实践。游戏不仅提供传统贪吃蛇的乐趣，更融入了**数学学习**、**成就激励**、**家长控制**等教育元素，让孩子在游戏中学习成长。

### 🎯 设计理念

- **教育性优先**: 寓教于乐，在游戏中自然学习数学
- **儿童友好**: 可爱角色、正向反馈、保护视力
- **家长安心**: 完整的家长控制系统，时间管理
- **安全第一**: 无广告、无内购、纯净游戏环境

## ✨ 核心功能

### 🎨 可爱角色设计
- **🐲 小火龙** (玩家1): 圆润头部 + 金色小角 + 渐变身体
- **🌈 彩虹蛇** (玩家2): 七彩渐变 + 星星装饰 + 闪亮效果
- **👀 大眼睛特效**: 高光点让角色更生动可爱

### 🍎 教育性食物系统
- **五种水果**: 苹果🍎、香蕉🍌、橙子🍊、葡萄🍇、糖果🍭
- **数学挑战**: 智能生成适龄数学题，答对获得额外奖励
- **难度分级**: 
  - 简单: 10以内加减法
  - 中等: 20以内四则运算
  - 困难: 50以内四则运算

### 🏆 成就激励系统
- **第一口美食** 🎉: 吃到第一个食物
- **数学小天才** 🧮: 答对10道数学题
- **小心翼翼** 🛡️: 5分钟内没有碰撞
- **超级大蛇** 🐍: 蛇身长度达到20节
- **速度之王** ⚡: 在困难模式下获胜

### 👨‍👩‍👧‍👦 家长控制面板
- **时间管理**: 15-60分钟可调节游戏时长
- **数学难度**: 三档难度适应不同年龄
- **数学频率**: 低频/适中/高频，平衡游戏与学习
- **功能开关**: 音效、鼓励消息可控制
- **智能提醒**: 友好的休息提醒，保护视力

### 🎯 智能频率控制
- **低频模式**: 偶尔出现数学题，专注游戏乐趣
- **适中模式**: 平衡游戏与学习，约每6-7个食物1题
- **高频模式**: 重点数学练习，约每4个食物1题
- **冷却机制**: 答题后8秒冷却，避免疲劳

## 🎮 游戏玩法

### 基础操作
- **玩家1 (小火龙)**: `W A S D` 键控制
- **玩家2 (彩虹蛇)**: `方向键` 控制
- **通用**: `空格键` 暂停/继续，`Enter` 重新开始

### 游戏模式
1. **单人模式**: 与智能AI对战
2. **双人模式**: 朋友面对面对战
3. **教育模式**: 数学题挑战模式

### 游戏目标
- 🍎 吃到更多食物获得分数
- 🧮 正确回答数学题获得额外奖励
- 🏆 解锁更多成就徽章
- 📚 在游戏中学习数学知识
